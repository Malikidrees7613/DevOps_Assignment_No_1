Coffee Shop Profit Predictor

A Streamlit-based application that forecasts coffee shop profitability using time-series and regression models.

Python Streamlit Data Science

Overview

This project demonstrates data ingestion, feature engineering, model training and a lightweight UI to let owners explore predicted revenue and sensitivity to factors like foot traffic and prices.

Highlights

  • Interactive Streamlit dashboard.
  • Feature engineering for seasonal and promotional effects.
  • Model explainability with SHAP values.
