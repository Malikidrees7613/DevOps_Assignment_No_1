# Portfolio Website Design Style

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a1a2e) and Charcoal (#16213e)
- **Accent**: Electric Cyan (#00d4ff) and Soft Purple (#6366f1)
- **Highlight**: Bright Orange (#ff6b35) for interactive elements
- **Text**: Pure White (#ffffff) and Light Gray (#e5e7eb)

### Typography
- **Display Font**: "Inter" - Modern, clean sans-serif for headings
- **Body Font**: "JetBrains Mono" - Monospace for code elements and tech feel
- **Accent Font**: "Space Grotesk" - Futuristic sans-serif for special elements

### Visual Language
- **Tech-Forward**: Modern, sophisticated AI/ML aesthetic
- **Clean Minimalism**: Plenty of whitespace with strategic color pops
- **Professional Edge**: Corporate-ready with creative flair
- **Interactive Depth**: Layered animations and hover effects

## Visual Effects & Animations

### Background Effects
- **Primary**: Animated particle system using p5.js creating floating data points
- **Secondary**: Subtle grid pattern with glowing nodes representing neural networks
- **Accent**: Aurora gradient flow in header areas

### Text Effects
- **Hero Title**: Typewriter animation with gradient color cycling
- **Section Headers**: Split-by-letter stagger animation on scroll
- **Code Elements**: Syntax highlighting with animated cursor
- **Interactive Text**: Color pulse and glow effects on hover

### Interactive Elements
- **Buttons**: 3D tilt effect with shadow expansion and color morphing
- **Cards**: Lift animation with depth shadow and subtle rotation
- **Images**: Ken Burns pan/zoom effect with overlay reveals
- **Navigation**: Smooth slide transitions with active state indicators

### Data Visualizations
- **Skills Radar**: Animated radar chart with progressive data reveal
- **Project Timeline**: Interactive timeline with expandable detail cards
- **Code Editor**: Simulated typing effects with syntax highlighting
- **Contact Form**: Real-time validation with smooth error/success animations

### Libraries Used
- **Anime.js**: Primary animation engine for smooth transitions
- **ECharts.js**: Interactive charts and data visualizations
- **Typed.js**: Typewriter effects for dynamic text
- **p5.js**: Creative coding for background particle systems
- **Splitting.js**: Advanced text animation effects
- **Matter.js**: Physics-based interactions for floating elements

### Scroll Motion
- **Reveal Animations**: Elements fade in with 20px upward translation
- **Parallax Effects**: Background elements move at 0.5x scroll speed
- **Progress Indicators**: Animated progress bars showing skill levels
- **Stagger Delays**: 100ms delays between related elements

### Hover Effects
- **Navigation**: Underline expansion with color transition
- **Project Cards**: 3D perspective shift with information overlay
- **Skill Icons**: Glow effect with surrounding particle burst
- **Social Links**: Bounce animation with color cycling

## Layout Structure
- **Hero Section**: Full viewport with animated background and typewriter intro
- **Skills Visualization**: Interactive radar chart with hover details
- **Project Showcase**: Timeline-based layout with expandable cards
- **Code Demonstration**: Live editor simulation with multiple language tabs
- **Contact Section**: AI-powered chat interface with form integration
